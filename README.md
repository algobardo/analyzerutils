# analyzerutils

## Desugar a project

- from source

```
pub run analyzerutils desugar --dir <prj dir> --entry <main entry file>
```
- from installation 
```
analyzerutils desugar --dir <prj dir> --entry <main entry file>
```

## Installing 

```
pub global activate -sgit https://github.com/algobardo/analyzerutils.git
```
